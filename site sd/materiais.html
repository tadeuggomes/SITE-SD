Componentes e Suprimentos

  • 1x Arduino Uno
    Descrição: É a placa microcontroladora que será programada para controlar os componentes do projeto. Com portas de entrada e saída (I/O), o Arduino Uno é uma escolha comum para projetos de eletrônica devido à sua simplicidade e versatilidade.
  • 1x Protoboard
    Descrição: Uma placa de montagem temporária usada para conectar componentes eletrônicos sem solda. Ela permite organizar e interligar componentes usando jumpers para criar circuitos com o Arduino.
  • 6x LEDs (2x verdes, 2x amarelos, 2x vermelhos)
    Descrição: LEDs são usados para indicar estados ou eventos. Os LEDs verdes podem representar um status de "ok" ou "pronto".
  • 3x Botões
    Descrição: Os botões são usados para interações do usuário. Quando pressionados, completam o circuito e enviam um sinal de entrada ao Arduino.
  • 1x Buzzer
    Descrição: Emite um som quando energizado. Ideal para avisos sonoros, como uma simulação de alarme no caso de uma sequência de botões incorreta.
  • 13x Jumpers
    Descrição: São fios usados para conectar os componentes na protoboard ao Arduino.
