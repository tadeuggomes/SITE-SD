Nome do projeto: B.O.M.B.A

Descrição do Projeto:

  • Motivação: Disponibilizar uma experiência interativa que incentive o uso de habilidades cognitivas de maneira divertida. Além de adquirir experiências voltadas ao Arduino e programação em C, para os desenvolvedores do projeto.
  • Objetivo: Criar uma simulação em que o usuário precisa pressionar os botões em uma sequência específica para desarmar a bomba. Visando alcançar um projeto de nível técnico aceitável e disponibilizar uma experiência relevante ao usuário final.
  • Contextualização: O projeto, desenvolvido no contexto das disciplinas de Sistemas Digitais e Introdução à Computação, visa aplicar conhecimentos de Arduino e programação em C para criar uma simulação. Além de desenvolver habilidades técnicas, o projeto busca oferecer uma experiência prática e envolvente, reforçando o interesse por sistemas embarcados e pela resolução de problemas usando tecnologia.
